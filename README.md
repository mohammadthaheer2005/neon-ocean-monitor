**NeonOcean V5.0** is an advanced **Hybrid Intelligence System** designed for real-time oceanographic monitoring, algae bloom prediction, and satellite data visualization. It integrates **Groq AI**, **NASA Satellite Feeds**, and **Bio-Digital Twin** technology into a single, futuristic command center.
